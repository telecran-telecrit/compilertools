Kimwitu is a system that supports the construction of programs that use trees or terms as their main data structure.

MEMPHIS

The Memphis Tree Builder & Tree Walker Tool supports definition and processing of abstract syntax trees. A precompiler extends C/C++: Domain declarations describe data types in a grammatical style. Match statements process these data using tree pattern matching.

TXL

A language for transformational programming. The basic paradigm of TXL involves transforming input to output using a set of transformation rules that describe how different parts of the input are to be changed in the output.
