Distribution

Memphis is freely available for noncommercial usage.

Memphis is distributed as platform independent C code. The installation files are currently prepared for Unix.

Download

    memphis-1.9.tar.gz

Unpacking

To unpack the distribution type 
   gunzip memphis-1.9.tar.gz
   tar -xvf memphis-1.9.tar
This creates a directory memphis with the following subdirectories
  • memphis
    The C code of the Memphis precompiler
  • mrts
    Contains a simple module to be linked with Memphis programs
  • examples
    Some small examples
  • doc
    The documentation

Installation

To install the Memphis precompiler go to directory memphis and type 
   build
You also need a small library function that has to be linked to your Memphis programs. To provide this go to directory mrts and type 
   build

Test

To test your installation go to directory examples/trees and type 
   build
