Bison is a general-purpose parser generator that converts a grammar description for an LALR(1) context-free grammar into a program to parse that grammar. The wrappers and support files in this package allow Bison to be used for the generation of parsers using Eiffel.

Features:

  • Based on GNU Bison version 1.25
  • Additional option "--Eiffel" for Eiffel Output
  • Multiple data types for semantic values (with type checking)

Enhancements:

  • Multiple passes
  • Default Actions are optional (--no-default-action)

Restrictions:

  • Semantic Parser not supported
